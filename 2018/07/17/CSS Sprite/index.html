CSS Sprite

2018-07-17

简介

css雪碧（精灵）是一种CSS图像合并技术，该方法是将小图标和背景图像合并到一张图片上，利用css的背景定位来显示需要的部分，即background-position：x y；

原理

把网站上需要用到的图片整合到一张单独的图片中，从而减少网站的http请求数量，其中，图片是在CSS中定义。

优点

1. 减少加载网页图片时对服务器的请求次数

  1. 提高页面的加载速度
